# Code of Conduct

This project adopts the [Rust Code of Conduct](https://www.rust-lang.org/policies/code-of-conduct)
verbatim. By participating in any space affiliated with this project — issues,
pull requests, discussions, or any other channel — you agree to abide by it.

## Reporting

For conduct concerns specific to this project, contact the maintainer at
**greg@echelberger.net**. Reports will be handled in confidence.

For broader Rust-community concerns, see the
[Rust moderation team's contact information](https://www.rust-lang.org/governance/teams/moderation).
